# Expense Tracker

**Автор:** [Ваше Имя и Фамилия]

## Описание
Простое GUI-приложение на Python для учета личных финансов. Позволяет добавлять расходы, указывать категории и даты, а также автоматически сохраняет данные в формат JSON.

## Функционал
- Добавление записи (Сумма, Категория, Дата).
- Валидация вводимых данных (защита от пустых полей и неверных форматов).
- Автоматический подсчет общей суммы.
- Хранение данных в локальном файле `expenses.json`.

## Как использовать
1. Установите Python 3.x.
2. Запустите файл: `python main.py`.
3. Введите сумму расхода, выберите категорию и нажмите "Добавить".
